# remote-ctrl

## 📡 remote-ctrl

基于 MFC 的远程控制工具，支持文件管理、远程监控、鼠标操作、锁机/解锁功能。

---

## 🧰 项目简介

`remote-ctrl` 是一个使用 C++ 和 MFC 构建的远程主机控制系统，主要功能包括：

- 📁 查看远程主机的文件
- 📥 远程下载文件
- 🗑️ 远程删除文件
- 📂 远程打开文件或文件夹
- 🖥️ 实时远程桌面监控
- 🖱️ 鼠标控制（移动、点击等操作）
- 🔒 锁定远程主机屏幕
- 🔓 解锁远程主机屏幕

---

## ⚙️ 技术栈

- **语言**：C++
- **框架**：MFC（Microsoft Foundation Classes）
- **开发环境**：Visual Studio 2019
- **平台支持**：仅限 Windows 10
